# Program Kalkulator Lingkaran

Halo semua kali ini saya membuat program kalkulator lingkaran, program ini dibuat karena adanya pelajaran saya disekolah yang membahas tentang lingkaran untuk mempermudah saya mengerjakan soal Lingkaran terciptalah program ini. Program ini dibuat menggunakan bahasa Python, saya menggunakan Python karena kemudahan dan simple aja kalau pakai Python

## Cara pakai

### Persyaratan
Anda harus menginstall program dibawah ini agar bisa berjalan
- Python
- Git (opsional jika anda download lewat zip/tar.gz langsung)

### Langkah Langkah
1. Pertama tama clone reository ini dulu dan pindah kedirectory foldernya
   ```
   git clone https://github.com/ohnolil/kalkulator-lingkaran.git
   cd kalkulator-lingkaran
   ```

2. Kemudian jalankan
   ```
   python main.py
   ```

Semudah itu
